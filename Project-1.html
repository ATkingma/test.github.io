About ​Level Generator

The LevelGenerator is part of an ongoing project by NoPressureStudios. I feel privileged to have contributed to this project, and I've gained valuable insights by studying their code. In this project page, I aim to showcase what I've learned and how much I've progressed. Please find below the best code I've developed for this project.

MapGenerator

This is a series of scripts that generate a map using pre-made chunks. I'd like to share with you the spawning methods I've used in a code snippet.The generation process begins by spawning a pre-set starting chunk. The chunk is then checked for anchors, and the anchors are checked for a parent chunk. From the direction of the anchor, the pre-set list is retrieved, and three random chunks are selected. The most preferred chunk is then chosen from these random options.Next, the newly-selected chunk is spawned at a position outside of the camera view. The anchor positions on both the parent chunk and the new chunk are checked, and a new position is calculated and set for the newly-spawned chunk. The chunk is added to a list of non-checked chunks, and the process is repeated with the same steps to progress further.
